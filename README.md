# Sanctus Gardens City

> **Investment Discovery Platform** — Heri Homes and Properties Limited  
> Ogbogoro, Port Harcourt, Rivers State, Nigeria

A premium static site for Africa's next visionary mega-development. 12 pages, zero build step, open in any browser.

---

## Live

[🌐 sanctusgardens.city](https://blvckpanda.github.io/sanctusgardens.city/)

## Pages

| Page | What it does |
|------|-------------|
| `index.html` | Homepage — hero with canvas particle system, vision, stats, districts |
| `masterplan.html` | Interactive SVG map — zoom/pan/hover/click |
| `district.html` | Dynamic district detail (`?id=residential`, `commercial`, etc.) |
| `opportunity.html` | Investment opportunities with gated info modal |
| `infrastructure.html` | City infrastructure — 7 systems + landmarks |
| `about-city.html` | Vision, phases, strategy |
| `about-heri.html` | Company profile & leadership |
| `contact.html` | Lead capture form |
| `sustainability.html` | Sustainability KPIs |
| `partnerships.html` | Partnership opportunities |
| `thank-you.html` | Post-submission confirmation |

## Stack

**HTML · CSS · JS** — no build tools, no bundler, no dependencies.

## Design

| Token | Value |
|-------|-------|
| **Headings** | Fraunces (300–600, italic) |
| **Body** | Inter (300–700) |
| **Primary** | Deep green `#0F3D2E` |
| **Accent** | Gold `#B8923A` |
| **Background** | Warm cream `#FCF8F0` |

## Run locally

```bash
python -m http.server 8000
# or
npx serve .
```

## Deploy

Static site — drag to Netlify, push to GitHub Pages, or drop on any web server.
